Welcome to Auto Test Runner!

Auto Test Runner is designed to run your unit and integration tests in the background. This allows you to develop faster and with the confidence you need.

How does this work?
  • First setup the API using this guide this doesn't exist yet
  • Then setup the windows service using this guide this doesn't exist yet
  • Once the site is running, add a project path to be watched
  • Build the test project, give it a few seconds (depending on how long it takes to run your tests) and you will receive a windows notification that will link you back to the site.
